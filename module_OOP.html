Welcome to the Object-Oriented Programming (OOP) module

In this module, we will use the Unified Modelling Language (UML) to support the design of OOP programs, and Python to implement these designs.

In this OOP module, we'll dive deeper into Python programming to learn the fundamentals of creating object-oriented programs. This covers essential concepts like abstraction, encapsulation, inheritance, and polymorphism.


A summary of the learning outcomes of the module:

• Develop an object oriented program using Python

• Prepare UML models to support the object oriented design process

• Apply data structures to support the storage of data in a variety of ways

• Implement data search algorithms to process stored data in the most efficient approaches

• Use design patterns to organise code contributing to an individual system

Reference: University of Essex Online, Object Oriented Programming November 2023. [online] Available at: https://www.my-course.co.uk/course/.
