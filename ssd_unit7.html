Unit overview:

Welcome to Week 7 where we will discuss Operating Systems (OS) and their relationships to programming languages and security. Most applications need to interact with the underlying operating system – whether it is to pick up input from a keyboard (or other input device) or to output data to a display. There are also security implications because such interactions often require the user application to link with software libraries – either supplied by the OS provider or a third party.


In this unit we shall:


  • Define the type and functions of a typical Operating System.
  • Give examples of common operating systems.
  • Explain the differences between processes, threads and schedulers.
  • Describe key approaches to make OSes more secure.
  • Discuss the difference between virtualisation approaches.
  • Advise on the differences between dynamic and shared libraries.

On completion of this unit you will be able to:


  • Describe mitigations to make operating systems more secure.
  • Explain some of the key risks and vulnerabilities associated with operating systems.
  • Outline the differences between static and dynamic libraries and when to use each.

Exploring a simple Python shell

Activity instructions:
In this session, you will create a command shell in Python, and then run it and answer questions about it.
Review the blogs at Praka (2018) and Szabo (n.d.) and then create a CLI/ shell that implements the following:

- When you enter the command LIST it lists the contents of the current directory
- The ADD command will add the following two numbers together and provide the result
- The HELP command provides a list of commands available
- The EXIT command exits the shell
Add suitable comments to your code and add the program to your e-portfolio.

Code of my program: 


								import os
								import cmd

								class MyShell(cmd.Cmd):
									"""Simple shell CLI."""

									# The command prompt string
									prompt = "$ "

									# Introduction message displayed at the start
									intro = "Welcome to MyShell! Type HELP to list commands."

									def do_LIST(self, inp):
										"""
										Lists the contents of the current directory.
										"""
										try:
											# List all items in the current directory and print them
											for item in os.listdir("."):
												print(item)
										except Exception as e:
											# Print an error message if an exception occurs
											print(f"Error: {e}")

									def do_ADD(self, inp):
										"""
										Adds two numbers together and print the result.
										"""
										try:
											# Split the input string to get the numbers
											parts = inp.split()
											if len(parts) != 2:
												# Print usage message if the number of parts is not 2
												print("Usage: ADD  ")
												return
											# Convert the input strings to float
											float1 = float(parts[0])
											float2 = float(parts[1])
											# Calculate the result
											result = float1 + float2
											# Print the result
											print(f"The result is: {result}")
										except ValueError:
											# Print an error message if the input values are not valid
											print("Error: Please enter valid numbers.")

									def do_HELP(self, inp):
										"""
										Displays the list of available commands.
										"""
										print("Available commands:")
										print("LIST - List the contents of the current directory")
										print("ADD   - Add two numbers and display the result")
										print("HELP - Display this help message")
										print("EXIT - Exit the shell")

									def do_EXIT(self, inp):
										"""
										Exits the shell.
										"""
										print("Goodbye!")
										# Return True to exit the cmd loop
										return True

									def default(self, inp):
										"""
										Handles unknown commands.
										"""
										# Print a message indicating the command is not recognized
										print(f"Unknown command: {inp}")

								if __name__ == '__main__':
									# Create an instance of MyShell
									shell = MyShell()
									# Start the command loop
									shell.cmdloop()
